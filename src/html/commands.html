Below lists all of the commands that you have made. You can add some here, or add them from chat. (Check Glimboi Commands)
Glimboi commands are default system commands. They are controlled by the rank system.
Words in all caps should be replaced with their value. Ex. NAME would be the name of the command.
Commands with parentheses() have multiple triggers. Ex. both !cmd new and !cmd add are the same thing.
Name Example Result Rank Perms
!commands !commands Returns a list of commands. None
!(cmd/command) (add/new) NAME DATA !cmd new !hello Hello there! Creates a new command. Variables supported. Add Commands
!(cmd/command) (del/delete/remove) NAME !command remove !hello Removes a command. Remove Commands
!(cmd/command) (help/info) !cmd info Retruns info about commands. None
!user (add/new) USER !user new Mytho Adds a user to Glimboi. Add Users
!user (del/delete/remove) USER !user del Mytho Removes a user from Glimboi. Remove Users
!quote !quote Returns a random quote. None
!quote (add/new) USER QUOTE !quote add Mytho Hellow World! Adds a new quote to the user specified. Add Quotes
!quote (del/delete/remove) USER QUOTE-ID !quote delete Mytho 5 Removes a quote from a specific user. Remove Quotes
!points !points Returns how many points the user has. None
!points top !points top Returns the user with the most points. None
!points NUMBER !points 5 Returns the user with the specified poition on the leaderboard. None
!points (add/inc/+) USER AMOUNT !points add Mytho 500 Adds points to a specific user. Add Points
!points (sub/dec/-) USER AMOUNT !points - Mytho 500 Removes points from a specifc user. Remove Points
!points (set/=) USER AMOUNT !points set Mytho 500 Sets a users points to a specific value. Edit Points
!points get USER !points get Mytho Returns the specified users points. None
!points help !points help Returns info about the point system. None
!rank !rank Returns the rank of the user. None
!song !song Returns the song that is currently playing. None
!sr (current/now) !sr now Returns the song that is currently playing. None
!sr (next/n) !sr next Returns the next song. None
!sr (last/l) !sr l Returns the last song. None
!sr skip !sr skip Plays the next song in the playlist (repeat dependent). Music Controls
!sr previous !sr previous Plays the last song in the playlist (repeat dependent) Music Controls
!sr (repeat/loop) !sr loop Toggles music loop setting. Music Controls
!sr (shuffle/random) !sr shuffle Toggles shuffle play setting. Music Controls
!sr toggle !sr toggle Plays or pauses the music. Music Controls
Variables are inserted into the action ChatMessage as $NAME . They are replaced with specific data as listed below.
