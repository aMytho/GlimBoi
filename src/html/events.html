Raffle

A raffle picks 1 user who entered to win a set amount of channel currency.

A raffle can be started on this page or by typing !raffle in chat. Users type !enter to join the raffle.

The results will be updated live on this page if it is open when the raffle starts. Leaving the page will remove the list, but the raffle is not affected. The winner will be announced to chat.

Gamble

Gamble is a way for users to win/lose points.

Start by typing !gamble wager. If they lose they lose all the wagered points. If they win they win the points based on the multiplier.

Multiplier Examples

1.0 => Gained the wager.

1.5 => Gained the wager and and extra half of it.

2.0 => Gained twice the wagered points.

Queue

An organized method for user actions. Ex. Joining a game

The queue can be started on this page or by typing !queue in chat. Users type !join to enter the queue. You can also add users with !queue add USER

The results will be updated live on this page. Commands below.

!queue : Starts the queue.

!queue stop : Stops the queue.

!queue reset : Resets the queue.

!queue add USER : Adds a user to the queue.

!queue remove USER : Removes a user from the queue.

!queue progress : Progresses the queue.

!queue next : Returns the next user in the queue.

!queue length : Returns the length of the queue.
